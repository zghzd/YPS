# 运行  
在64位Windows命令提示符中，使用此指令`yps [--<key>-<value>] [...]`启动并指定一些参数。  

在控制台参数中：  
- 格式：`--key-value`  

[参数详情](ArgumentValue.md)  

# 下载源代码并重新构建  
## 准备步骤  
- IDE:Microsoft visual studio 2022
- 克隆此存储库
- 准备public.xml到D:/YPS/x64/目录下
- 使用release模式完整构建
- 之后，在D:/YPS/build and package/目录下，可以找到构建并打包完成的zip文件
